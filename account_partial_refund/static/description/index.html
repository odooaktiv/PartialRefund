Partial Refund

The Partial Refund module will provide an option of generating a refund invoice directly without going to the draft state for editing the invoice. The default Odoo module when clicked on the refund button generated a bill similar to original bill in a draft state. If user wants to change the amount for quantity of products to be refunded then they need to go to the draft state and edit that. While we have provided the facility to edit the refund invoice directly when initiated for the refund.

Prerequisites before Installation

No prerequisites are required before the installation.

Configurations

No configurations are required.

Let’s consider creating an invoice for a product with 10 quantity.

But now user wants to change their required number of quantities. They want to reduce the quantity of the product. By clicking on “Refund Invoice” user can edit the quantities.

Before proceeding to the refund draft, user want to change the number of quantities. The required quantity field is editable now so let us consider changing the quantity from 10 to 5.

You can see in the below image, after editing the quantity of the product a refund invoice is generated updating the amount to be refunded for the rest 5 products. As in this case we have mentioned 10 products users can get the refund for all of the 10 products but not for more than 10.
